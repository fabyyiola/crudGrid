#crudGrid
This plugin is to make developers life easy when it comes to create a catalog to insert, update, delete records.

##Options:

  >**Mapping**
  - dataField: string
  - colText: string
  - [visible: bool]
  
| Level 1 | Level 2 | Level 3 |
| :---         |     :---:      |          ---: |
| git status   | git status     | git status    |
| git diff     | git diff       | git diff      |
